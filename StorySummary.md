# ストーリー (1)

# 第一章：崩れる山の記憶（くずれるやまのきおく）

中学生のプレイヤーは野外授業中に山の崩落に巻き込まれ、それをきっかけに地質研究機関G-Labと接触し、研究員カエデに導かれて秘密施設へと招かれる。カエデがプレイヤーに地層サンプルの採取と分析の任務を無理やりさせた。調査の結果、古代断層の活動再開を示す異常が確認され、プレイヤーは次なる任務へと向かう。

## 登場人物

- **主人公（プレイヤー）**：中学生。遠足活動のとき事故があった。それをきっかけとして、G-Lab（Geography-Lab）の一員となる。
- **Dr.Kaede（Kaede）**：G-Lab所属の謎の研究員。プレイヤーの指導担当。
- **教師**：登山遠足の引率。
- **研究員A**：G-Labの通信担当。後半に登場。

---

## シーン1：遠足中の出来事

- 遠足中、山の一部が突然滑落。主人公が崩落現場で奇妙な岩石を発見。
- 謎の通信が入り、G-Labに接触される。

```
主「うわっ！？ ちょっと待って、足場が崩れた！」
教師「◯◯くん、大丈夫！？ 落ちないで！」
		「みんな！岩の近くには近寄らないように！」
主「……なんとか無事みたいだけど……この岩、なんだか層がくっきりしてる……？」
（近くに落ちていた通信端末が起動し、音声が流れる）
？？？「こちらG-Lab。現場近くに設置していた観測ドローンが地殻信号の異常を検知しました。そちらに急行中です。」
```

```
主「……すごい、まるで秘密基地みたいだ。」

カエデ「君が今から目にするのは（Labで放置している採集設備とか）、地球の“内部”と対話するための装置たちだ。ようこそ、G-Labへ。」
```

## シーン3：G-Lab研究室での初対面

- カエデ登場。G-Labと地殻異変について説明。
- 初任務「地層調査」発令。

```
【研究員・カエデ】（正式登场）
「ようこそ、G-Labへ。私は地質災害予測班のカエデ。今後、君の指導を担当する。」

【主人公】
「え...指導担当って...?」

【カエデ】
「私たちは、地球内部で起きる変動——断層活動、岩層の不整合、未知の地殻異常を追跡している。今回の滑落事件もその一つだ。」

【主人公】
「私の話聞いている？」

【カエデ】
「よーし、まずは基礎から始めよう。」

【カエデ】
「先日の滑落地点には、年代の異なる3層の地層が露出している。その中に“地質異常サンプル”が含まれていると見られる。」

【主人公】
「つまり、見た目じゃわからないけど、何かヤバい岩が混じってるってことか……」

【カエデ】
「そういうこと。これは君の初任務だ。必要なツールはすでに用意してある。」

```

**【ゲーム提示】**

任務「地層試料を採取せよ」が開始されました。

取得ツール：地質ハンマー(最初は薄片のサンプルを採取でいいから)
