# 样本切割系统交互指南

## 🎮 新的交互流程

### 1. 接近实验台
- 玩家需要走到实验台附近（3米范围内）
- 会出现屏幕下方的交互提示：**"[F] 使用样本切割台"**

### 2. 打开切割界面
- 按 **F键** 打开全屏切割界面
- 界面分为两部分：
  - **左半边**: 样本仓库界面（显示你的地质样本）
  - **右半边**: 切割操作区域（绿色拖拽区域）

### 3. 选择样本
- 在左侧仓库界面中找到 **多层样本**（通常有多种颜色层）
- **点击并拖拽** 样本到右侧的绿色区域

### 4. 开始切割
- 拖拽成功后会启动切割小游戏
- **红色切割线** 在样本上移动
- **绿色区域** 是成功切割区域
- 在绿色区域内按 **空格键** 进行切割

### 5. 完成切割
- 切割成功后会生成多个单层样本
- 单层样本会自动添加到你的背包中
- 可以用于后续的矿物分析

## ⚠️ 注意事项

### 样本要求
- 只有 **多层地质样本** 才能进行切割
- 单层样本无需切割
- 必须是通过钻探工具获得的样本

### 切割难度
- **2层样本**: 40cm 成功区域（较容易）
- **3层样本**: 20cm 成功区域（中等）
- **4+层样本**: 10cm 成功区域（较难）

### 操作技巧
- 观察切割线的移动节奏
- 提前准备按键，在绿色区域快要到达时按空格
- 切割失败会 **销毁整个样本**，请谨慎操作

## 🔧 故障排除

### 问题：按F键没有反应
- 检查是否在实验台3米范围内
- 确保切割台已正确集成到场景中

### 问题：拖拽不起作用
- 确保选择的是多层样本
- 检查样本是否在仓库中（不是在背包中）
- 尝试重新打开切割界面

### 问题：切割线不移动
- 确保样本已成功拖拽到切割区域
- 检查系统初始化是否完成

## 🎯 使用建议

1. **先准备样本**: 使用钻探工具采集多层地质样本
2. **选择合适时机**: 在Laboratory Scene中使用切割台
3. **练习时机**: 多练习切割时机的把握
4. **保存进度**: 切割前建议保存游戏进度
